## What is this?

A simple timer that helps you maximize cache hits when building with [Emergent.sh](https://emergent.sh). 

When you send code or context to Emergent, it stays cached for approximately 5 minutes. Any follow-up tasks within that window cost significantly fewer credits. This tool helps you track that window and batch your tasks efficiently.

---

## How it works

1. **Prepare 3-5 related tasks** before starting
2. **Click "I Sent Context"** when you send code/context to Emergent
3. **Complete your tasks** while the cache is hot
4. **When timer expires:** Push to GitHub, start a fresh chat, import your repo
